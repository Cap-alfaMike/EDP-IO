# EDP-IO Java REST API

This directory contains the Java Spring Boot REST API for the EDP-IO platform.

## Quick Start

```bash
cd edp-io-api
mvn clean package
java -jar target/edp-io-api-1.0.0.jar
```

API runs on `http://localhost:8080/api`

API Docs: `http://localhost:8080/swagger-ui.html`

See [README.md](./README.md) for detailed documentation.
